Retail Sales Analytics & Business Intelligence Project
Overview

This project focuses on analyzing retail sales performance to uncover actionable insights related to sales trends, product performance, customer behavior, and profitability. Using a combination of Python, SQL, Excel, and Power BI, the analysis translates raw transactional data into decision-ready business intelligence.

Tools & Technologies

Python: NumPy, Pandas, Matplotlib, Seaborn

MS Excel: Data validation and exploratory summaries

MySQL: Business-focused SQL queries and aggregations

Power BI: Interactive dashboards and KPI visualization

Project Objective

Analyze historical retail sales data to identify revenue and profit trends.

Determine top-performing and underperforming products.

Understand customer purchasing behavior across regions and segments.

Identify key profit drivers and loss-making categories.

Deliver visual, executive-friendly insights to support strategic decisions.

Key Features

Data Cleaning & Feature Engineering

Handling missing values and duplicates

Creating derived metrics such as profit margin and sales growth

Exploratory Data Analysis (EDA)

Trend analysis by time, category, and region

Customer and product-level performance analysis

SQL-Based Business Queries

Sales, profit, and customer segmentation queries

Region-wise and category-wise performance analysis

Power BI Dashboard

Interactive KPIs for sales, profit, and order volume

Drill-down views by category, region, and customer segment

Dataset

Source: SuperStore Orders Dataset

Data Type: Retail transactional data

Key Fields: Order Date, Product Category, Sales, Profit, Customer Segment, Region

Business Impact

Enables identification of high-revenue and high-profit product categories.

Highlights loss-making products and regions requiring corrective action.

Supports inventory planning, pricing strategy, and customer targeting.

Provides a scalable BI framework for ongoing retail performance monitoring.

Deliverables

Cleaned and analyzed datasets

SQL queries for business insights

Interactive Power BI dashboard

Business-ready analytical summary

Future Enhancements

Predictive sales and demand forecasting

Customer lifetime value (CLV) analysis

Automated Power BI data refresh

Advanced segmentation using clustering techniques
